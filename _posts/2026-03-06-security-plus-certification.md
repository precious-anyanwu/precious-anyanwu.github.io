Overview

I am excited to share that I successfully passed the CompTIA Security+ certification exam with a score of 782.

![Security+ Certification](assets/images/security-plus/security-plus-pass.jpeg)

Security+ is a globally recognized cybersecurity certification that validates foundational knowledge across:

Threat detection and response

Network security

Identity and access management

Security architecture

Cryptography and PKI

Security operations and incident response

This milestone represents a significant step in my journey toward becoming a Security Operations Center (SOC) Analyst.

My Cybersecurity Learning Journey

My preparation for Security+ built upon several years of structured learning and hands-on practice in cybersecurity, networking, and cloud security.

Prior to this certification, I obtained the ISC2 Certified in Cybersecurity, which provided a strong introduction to core security principles.

To reinforce theoretical knowledge, I focused heavily on practical labs and investigation-based learning, documenting my work throughout the process.

Some of the practical areas I worked on include:

Malware traffic analysis using packet captures

Network packet investigation with Wireshark

Log analysis using tcpdump

Threat intelligence lookups using VirusTotal

Identifying Indicators of Compromise (IOCs)

Network scanning and anomaly detection

These hands-on exercises helped translate certification concepts into real-world security investigation workflows.
